UVA metro

Congress Apportionment
I created an application that calculated the correct apportionment ( the number of seatings in the congress ) for each state. There were three historical apportionment strategies, Hamilton, Jefferson, and Hungintonhill, that were implemented. The app showed the pros and cons of each strategies. I read in CSV and excel file using Java and used top-down and factory implementation to build this application. Through this project, I was able to learn how to organize and structure multiple modals and classes to handle large number of datas and write code that is easily maintainable and loosely coupled.
